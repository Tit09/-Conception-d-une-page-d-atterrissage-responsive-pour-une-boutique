Dja Shop - Page d’atterrissage responsive
Description

Ce projet est une page d’atterrissage pour une boutique en ligne réalisée avec HTML, CSS et Bootstrap.

L’objectif est de créer une interface simple, moderne et responsive qui s’adapte à tous les types d’écrans (mobile, tablette, ordinateur).
Fonctionnalités:

- Barre de navigation responsive
- Section d’accueil avec image de fond
- Section produits avec cartes Bootstrap
- Formulaire de contact
- Pied de page
- Navigation fluide entre les sections

 Technologies utilisées

- HTML5
- CSS3
- Bootstrap 5
Responsive design
Le projet utilise le système de grille de Bootstrap pour s’adapter aux différentes tailles d’écran :

- 4 produits sur grand écran
- 2 produits sur tablette
- 1 produit sur mobile

- Installation:
  1. Cloner le projet :
